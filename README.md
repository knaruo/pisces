# pisces
Pisces - Tiny Split Keyboard

![Pisces V1](doc/res/pisces_v2.png)


# Concept

* Ortholinear
  * 3 rows \* 6 columns + 1 thumb key, total 38 keys
* Key switch hot-swap capable
  * Supports Kailh PCB socket for MX
* Thin design
  * Only supports SMT diodes (1N4148W)
